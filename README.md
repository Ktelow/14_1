# Project 2

## Описание:
Project 2 - проект по созданию функционала для работы с темой E-commerce

## Требования к окружению:
   ```
   Python 3.12+
   ```

## Установка:

1. Клонируйте репозиторий:
    ```
   git clone [https:](https://github.com/Ktelow/14_1)
    ```
2. Установите зависимости:
   ```
   pip install -r requirments.txt

3. Создать и активировать виртуальное окружение:
   ```
   python -m venv venv
   venv\Scripts\activate

## Тестирование:
Этот проект тестируется с помощью фреймворка pytest
Для работы с ним необходимо выполнить команду:
   ```
   poetry add --group dev pytest
   ```
И далее можете выполнять тесты с помощью команды:
   ```
   pytest --cov
   ```

## Использование:

Добавлены классы для создания продуктов и их категорий, а также функции подсчёта количества продуктов и их категорий
